ABOUT LICT321

This course provides an in-depth exploration of web technologies and their applications in libraries and information centers. Web technology has grown to include a variety of business, academic, organizational, and social applications. Diverse multicultural and multilingual user communities now depend on web technology. Students will learn about the tools and apply modern development methodologies and practices for web technologies to enhance information access, management, and services. The course offers practical, hands-on discussions and exercises in utilizing various web technology languages and tools.

"In the age of information, librarians are navigators, not gatekeepers."

Anne M. Warner

The World Wide Web specifically refers to information resources.
The World Wide Web is a virtual network of websites connected by hyperlinks (or "links"). Web sites are stored on servers on the internet, so the World Wide Web is a part of the internet.

How the Web Works
  • User selects a target URL
  • Browser takes the URL, embeds it inside an HTTP request, and sends it to the server.
  • The HTTP server receives the request on the port, makes a socket connection.
  • The server finds the requested HTML file and concatenates it with status info.
  • The server ships back the file, and closes the connection
